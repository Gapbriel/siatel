Equipos para Telefonía Pública

CT 8016

Control integral para cabinas públicas y cyber

  • Control para 16 Cabinas telefónicas.
  • 16 Pc´s en forma simultánea.
  • Impresora Térmica incorporada.

CT 8002 GSM

Controlador Telefónico Móvil

  • Controla hasta dos líneas GSM (puede ser sub-equipado con una línea celular).
  • Modalidad Post-pago y Pre-pago.
  • Dos entradas RJ11 para líneas de par de cobre.
  • Impresora Térmica incorporada.

CT 8002 (Cobre)

Ideal para dos cabinas de telefonía pública

  • Controla hasta dos líneas de par de cobre
  • Impresora Térmica incorporada.

Visor

Ideal para dos cabinas de telefonía pública

  • Visor de cabina para toda la línea de tarifadores CT
  • Apto para funcionar con Soft Wincab 6016 y 8016.
